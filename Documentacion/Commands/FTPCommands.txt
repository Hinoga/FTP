Innitial FTP list of commands

+-------------------------------------------------------------+
|                       List of commands                      |
+-------------------------------------------------------------+
| FTP command | Description of the command                    |
+-------------+-----------------------------------------------+
| ls          | Lists the content of the directory            |
+-------------+-----------------------------------------------+
| disconnect  | terminate ftp session                         |
+-------------+-----------------------------------------------+
| exit        | terminate ftp sessions and exit               |
+-------------+-----------------------------------------------+
| get         | receive file                                  |
+-------------+-----------------------------------------------+
| help        | display local help information                |
+-------------+-----------------------------------------------+
| connect     | connect to remote ftp                         |
+-------------+-----------------------------------------------+
| user        | send new user information                     |
+-------------+-----------------------------------------------+
| mget        | Copy multiple files from the remote directory |
+-------------+-----------------------------------------------+
